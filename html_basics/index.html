Conceptos basicos de Html

Por buena practica solo deberia haber un h1 por vista, usar los siguientes para organizar ideas segun jerarquia

Algunos elementos basicos:

Parrafo p

Esto es un parrafo html, se utiliza para escribir bloques de texto, el p es un elemento de bloque por lo cual se verá como tal, puedes incluir elementos inline dentro de este elemento de bloque como por ejemplo las etiques b, i o u, estas etiquetas tienen estilos predefinidos que alteran como se ve el texto.
Puedes usar la etiqueta br para generar un salto de linea.
